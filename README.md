# Student Academic Performance Analysis

## Overview
This project performs an Exploratory Data Analysis 
on a student performance dataset of 2000 students 
across 7 subjects.

## Dataset
Source: Kaggle — mexwell/student-scores
Rows: 2000
Columns: 17

## Libraries Used
- Pandas
- Numpy
- Matplotlib
- Seaborn

## Key Findings
- Study hours most positively impacts performance
- Absences negatively affect scores
- Gender has minimal impact on performance
- Math scored highest among all subjects

## Tools
Google Colab
Python 3
